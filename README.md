### Gradle installation

Add your Maven repository:

```gradle
repositories {
    maven { 
        url = "https://api.modrinth.com/maven"
     }
}
```

Add the dependency:

```gradle
dependencies {
    implementation "maven.modrinth:teallib:<version>"
}
```

Replace `<version>` with the version of the library you wish to use.
